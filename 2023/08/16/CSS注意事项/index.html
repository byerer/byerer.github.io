CSS注意事项

技术规范

class 和 id

id 是一个在 HTML 文档中唯一标识一个元素的属性。使用 id 选择器可以选择具有特定 id 值的元素，并为其应用样式。id 适合用于标识页面中唯一的元素，比如页眉、页脚、侧边栏等。id 选择器使用 “#” 符号后跟 id 值来选择元素，例如 “#header”。

class 是一个可以在 HTML 文档中多次使用的属性，用于标识一组具有相似特征的元素。使用 class 选择器可以选择具有特定 class 值的元素，并为其应用样式。class 适合用于标识多个元素，并为它们应用相同或相似的样式。class 选择器使用 “.” 符号后跟 class 值来选择元素，例如 “.button”。

CSS书写顺序

根据chatgpt的答案
在编写 CSS 样式时，一般遵循以下常见的书写顺序：

  1. 布局相关属性：包括 display、position、float、clear 等属性，用于控制元素的布局和定位。

  2. 盒模型相关属性：包括 width、height、padding、margin、border 等属性，用于定义元素的尺寸和边距。

  3. 排版相关属性：包括 font、line-height、text-align、text-transform、white-space 等属性，用于控制文本的样式和排版效果。

  4. 背景相关属性：包括 background-color、background-image、background-position、background-repeat 等属性，用于设置元素的背景样式。

  5. 边框相关属性：包括 border-color、border-width、border-style 等属性，用于设置元素的边框样式。

  6. 动画和过渡相关属性：包括 animation、transition 等属性，用于添加动画效果和过渡效果。

  7. 其他属性：包括 color、opacity、box-shadow、text-decoration 等其他属性。

当然，这只是一种常见的书写顺序，并不是强制规定。你可以根据自己的喜好和项目需求进行调整。另外，为了提高代码的可读性，可以使用缩进、注释和空行等方式来组织和分隔不同的样式规则。
