声明对象

  1. 优先使用 const
  2. 如果对象需要变化，则使用 let
  3. 不要使用 var

DOM的基本认识

DOM

D 指 document ：当创建了一个网页并把它加载到Web浏览器中时，DOM就在幕后悄然而生了。它会将你编写的网页文档转换成为一个文档对象。

O 指 object ：指的是文档中的各个部分和元素。文档中的每个元素、属性、文本等都被看作是一个对象，这些对象可以通过DOM进行访问和操作。

M 指 model ：指的是DOM的结构化树状模型。DOM将文档解析为一个由节点和对象组成的树形结构，每个节点都有不同的类型和属性。

DOM树

DOM树（Document Object Model Tree）是指将HTML、XML或XHTML文档解析后生成的一种树形结构，用于表示文档的层次结构和关系。

获取元素内容

可以使用 innerHTML 或者 innerText

innerHTML和innerText是两个常用于操作元素内容的属性，它们有以下区别：

  1. 内容处理方式不同：

    • innerHTML：返回或设置元素内部的HTML内容，包括HTML标记。可以用于获取或修改元素内部的HTML结构，可以添加、修改或删除元素中的HTML元素和文本内容。
    • innerText：返回或设置元素内部的文本内容，会自动移除HTML标记。只获取或设置元素内的纯文本内容，会忽略HTML标记，只显示文本内容。

  2. 影响性能不同：

    • innerHTML：由于可以操作HTML结构，涉及解析和重新渲染，所以性能相对较低。如果需要频繁更新大量HTML内容，可能会有一定的性能影响。
    • innerText：仅涉及文本内容的操作，性能相对较高。如果只需要获取或设置文本内容，而不需要涉及HTML结构的操作，推荐使用innerText。

  3. 对隐藏元素的处理不同：

    • innerHTML：可以获取或设置隐藏元素的HTML内容，不受元素的可见性影响。
    • innerText：无法获取或设置隐藏元素的文本内容，只能获取或设置可见元素的文本内容。

attention：使用innerHTML可能会导致跨站脚本攻击（XSS）的风险。
